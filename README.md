# Mango · Prepared Landing Page

A bespoke, single-page landing built in Mango's own visual language: dark-first nav, Mango yellow pill CTAs, big orange and yellow feature tiles on 30px radius, Spanish-first headline with English body, short builder-to-builder sentences. The page carries the fit-analysis body for Mango, frames the BBVA explainability hypothesis, maps the VigilOre (Armetour) case to Mango's shape, and points at the Ethics Engine paper as evaluation cover for a credit engine under bank scrutiny.

Primary CTA routes to Matt Creamer's calendar (https://calendly.com/thecro-eduba/30min). Logo is local (`logo.png`), fonts load from Google Fonts (Manrope for display, Inter for body, matching Mango's production stack minus the licensed Tasa Orbiter). Eduba branding appears only in the footer.
